## Package install
install any dependencies

    ```bash
        yarn workdpace web add -D tailwindcss postcss autoprefixer
    ```
tailwind config initialization :

    ```bash 
        yarn workspace web tailwindcss init -p
    ```

Build projects :

    ```bash
        yarn build  
    ```

run project on server : 

    ```bash
        yarn workspace web start
    ```
